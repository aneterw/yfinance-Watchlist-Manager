# yfinance 自選股管理器

功能完整的股票/指數/ETF/外匯/加密貨幣 Watchlist 管理工具，支援 K 線圖、價格提醒、多語系、多主題、自訂字體。請注意Yahoo財經的市價指標通常會遲延15到20分鐘。資料僅供參考，不得用作任何投資決策之基礎。數據僅供參考，不應作為任何投資決策的依據。

---

## 1. 安裝與啟動

### 1.1 安裝相依套件

```bash
pip install -r requirements.txt
```

或單獨安裝：

```bash
pip install yfinance pandas matplotlib
```

Python 版本需求：Python 3.8 以上。

### 1.2 啟動程式

```bash
python watchlist_app.py
```

啟動後會看到圖形界面（GUI）。

### 3.2 新增 / 刪除 Watchlist

**新增：**

1. 點擊上方工具列的 **「＋ 新增 Watchlist」**
2. 輸入新清單名稱（如「我的自選股」）
3. 按 Enter 或點擊 OK

**刪除：**

1. 在左側點擊要刪除的 Watchlist
2. 點擊 **「－ 刪除 Watchlist」**
3. 確認對話框，選擇 Yes

---

### 3.3 加入商品到 Watchlist（3 種方式）

#### 方式 A：搜尋加入（推薦）

1. 先在左側選擇一個 Watchlist
2. 在右下方的搜尋框輸入 ticker 或公司名稱（如 `AAPL`、`Tesla`、`Bitcoin`、`2330`）
3. 搜尋結果即時顯示在下方列表（輸入字串時自動篩選）
4. **直接雙擊** 搜尋結果中的項目，就會加入目前選中的 Watchlist

> 搜尋功能使用本機股票索引（10,390+ 熱門股票），**離線也能使用**，不依賴網路。

#### 方式 B：按鈕加入所選

1. 搜尋後，用滑鼠點擊選擇搜尋結果
2. 在「顯示名稱」欄位輸入你想要的顯示名稱（可空白）
3. 點擊 **「＋ 加入所選」**

#### 方式 C：直接輸入 Ticker

1. 點擊 **「直接輸入 Ticker」** 按鈕
2. 在彈出的視窗中輸入 ticker（如 `BTC-USD`、`005930.KS`、`601318.SS`）
3. 可選填顯示名稱
4. 按 Enter 或點擊「加入」

> 常見 ticker 一覽：

### 3.5 更新價格

#### 更新目前 Watchlist

**方法：** 點擊上方工具列的 **「更新目前 WL」**

- 只更新目前選中的 Watchlist 內的商品
- 不影響其他 Watchlist 的價格

#### 更新所有價格

**方法：** 點擊上方工具列的 **「更新所有價格」**

- 更新所有 Watchlist 內的所有商品
- 程式會**背景 threading 執行**，不會卡住視窗
- 狀態欄會顯示「正在更新…請稍候」
- 更新完畢後，右側表格會即時刷新：
  - 漲：顯示紅色
  - 跌：顯示綠色
  - 平：深灰色

價格欄位說明：

| 欄位  | 說明                                    |
| --- | ------------------------------------- |
| 現價  | 目前市場價格（Yahoo Finance 報價可能延遲 15-20 分鐘） |
| 漲跌  | 今日漲跌金額                                |
| 漲跌幅 | 今日漲跌幅（%）                              |
| 成交量 | 今日成交量                                 |

### 3.6 自動更新

程式會**閒置 10 分鐘自動更新所有價格**，無需手動操作。

### 3.7 K 線圖（雙擊商品）

**操作方法：** 在「商品價格」列表中，**雙擊** 任一行商品，即可開啟 K 線圖視窗。

K 線圖視窗包含三個子圖：

1. **K 線圖**（上）：紅漲綠跌，顯示 OHLC 價格走勢
2. **成交量**（中）：柱狀圖顯示交易量（以最高量正規化）
3. **KD 指標**（下）：K 線（橘色）與 D 線（藍色），輔助買賣信號

週期切換：

- **日線**：近 2 年每日數據
- **週線**：近 10 年每週數據
- **月線**：近 30 年每月數據

### 3.11 價格提醒（⚙ 設定提醒）

**設定方式：**

1. 在商品列表選取一個商品
2. 點擊工具列 **「⚙ 設定提醒」**
3. 輸入高價提醒（高於此價格拉通知）和/或低價提醒（低於此價格拉通知）
4. 點擊 **「儲存」**

**觸發行為：**

- 當價格**高於**設定的高價提醒時，發送 Windows 系統通知（BalloonTip），顯示「🚨 高價提醒：{ticker} 已漲至 {price}」
- 當價格**低於**設定的低價提醒時，發送「📉 低價提醒：{ticker} 已跌至 {price}」
- 通知維持 12 秒自動消失

**管理提醒：**

- 選單可查看目前已設定的所有提醒
- 可刪除所選提醒或重置已觸發的狀態

## 4. 常見問題 FAQ

### Q：可以一次更新單一 Watchlist 嗎？

A：可以。點擊上方工具列的 **「更新目前 WL」** 只更新目前選中的 Watchlist；點擊 **「更新所有價格」** 則更新全部。

### Q：資料會永久保存嗎？

A：會的。程式會自動將 Watchlist、語言、主題、字體、提醒設定儲存在 `watchlist_data.json`，下次啟動自動恢復。

### Q：搜尋不到我要的 ticker 怎麼辦？

A：

1. 改用 **方式 C（直接輸入 Ticker）**，在對話框手動輸入官方 ticker 代碼
2. 常見格式：
   - 美股：`AAPL`、`GOOGL`
   - 加密貨幣：`BTC-USD`、`ETH-USD`
   - 期貨：`GC=F`（黃金）、`CL=F`（原油）
   - 台股：在 Yahoo Finance 上是 `2330.TW`
   - 港股：`0700.HK`
   - A 股：`601318.SS`（上海）、`000858.SZ`（深圳）
   - 匯率：`EURUSD=X`

### Q：更新價格很慢或失敗？

A：

- yfinance 需要連接網路，請確認網路連線
- Yahoo Finance 偶有流量限制，若短時間大量抓取可能被暫時封鎖，稍後再試即可
- 程式內建熱門股票索引（10,390+ 檔），搜尋功能**離線也能使用**

### Q：K 線圖無法顯示？

A：

- 請確認已安裝 `matplotlib`（`pip install matplotlib`）
- 部分加密貨幣或外匯資料較少，K 線圖可能顯示空白

### Q：暗黑模式下某些文字看不清？

A：暗黑模式已針對對比度優化，若仍有問題可嘗試切換回 System 或 Light 主題，或調整字體大小。

### Q：如何自訂字體？

A：選單列 → **Font** → 選擇 **Font Family** → 從系統已安裝字體中選擇；或調整 **Size** 改變字體大小（10-20）。

## 5. 技術架構說明

| 元件                        | 說明                              |
| ------------------------- | ------------------------------- |
| `yfinance`                | 向 Yahoo Finance API 取得即時股價與財務資料 |
| `tkinter` / `ttk`         | Python 內建 GUI 框架                |
| `matplotlib`              | K 線圖、成交量、KD 指標繪圖                |
| `threading`               | 背景抓價，避免介面卡死                     |
| `top250_tickers.json`     | 10390+ 熱門股票本地索引，搜尋功能離線可用        |
| `watchlist_data.json`     | 自動儲存所有設定與資料                     |
| `subprocess` + PowerShell | Windows 系統通知（BalloonTip）        |
